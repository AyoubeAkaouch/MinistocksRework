What is the stock code for 'abc'?

Stocks symbols can be found on the Yahoo Finance web site. If the stock symbol cannot be find on the site, then it will not work in the app.

back to top

Why does this stock code 'xyz' not work?

If the code exists on the Yahoo Finance website (see What is the stock code for? question) but it does not work in the app, then it is possible that the specific stock code has been disabled in the Yahoo data feed. The app uses a data feed provided by Yahoo that does not includes all the codes available on the website. There is usually nothing that I can do about these missing stock codes.

Note: Stock symbols from the following country's exchanges are known not to work:

  • Spain
  • India
